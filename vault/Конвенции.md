# Конвенции frontend

## Зона ответственности

Frontend-агент работает ТОЛЬКО в `work-together-front/`.

- ❌ Не трогать `WorkTogetherBack/`
- ❌ `docs/` — только чтение
- API-контракт задан backend'ом (`docs/PROJECT_SPEC.md` раздел 6 + `WorkTogetherBack/vault/API.md`) — потреблять как есть

## Правила кода

1. Поток: Page → Hook → Service → api.ts; не звать axios из компонентов напрямую
2. Токены — только in-memory token-store; localStorage запрещён для auth
3. `withCredentials: true` не убирать — сломается refresh
4. Все маршруты через `withLocale(locale, path)`
5. Каждая новая строка UI — в `messages/types.ts` + `en.ts` + `uk.ts`
6. `"use client"` — только где нужны хуки/интерактив
7. UI-киты (shadcn, MUI) не добавлять — Tailwind + свои `ui/`-компоненты
8. next-intl не добавлять — есть своя i18n
9. Zod для всех форм; лимиты полей = backend-лимиты
10. Query keys — только из `lib/query/keys.ts`; после мутаций инвалидировать
11. Файловые URL рендерить через `resolveFileUrl()`
12. `npm run check` и `npm run build` должны проходить
