# Маршруты (app/[locale])

Локали: `en` (default), `uk`. Middleware редиректит `/x` → `/{locale}/x`, cookie `wt_locale`.

## Публичные

| Маршрут                                | Что                               |
| -------------------------------------- | --------------------------------- |
| `/[locale]`                            | лендинг                           |
| `/[locale]/how-it-works`               | как это работает                  |
| `/[locale]/terms`, `/[locale]/privacy` | юридические страницы              |
| `/[locale]/projects`                   | каталог проектов + фильтры        |
| `/[locale]/projects/[id]`              | детали проекта (`isOwner` из API) |
| `/[locale]/students`                   | студенты, ищущие команду          |
| `/[locale]/users/[id]`                 | публичный профиль                 |

## Auth

| Маршрут                          | Что                                              |
| -------------------------------- | ------------------------------------------------ |
| `/[locale]/auth/login`           | вход (+ OAuth кнопки)                            |
| `/[locale]/auth/register`        | регистрация → «проверьте почту», без автологина  |
| `/[locale]/auth/confirm-email`   | подтверждение email по токену                    |
| `/[locale]/auth/callback`        | финиш OAuth: refresh по cookie → access в память |
| `/[locale]/auth/forgot-password` | заглушка                                         |

## Защищённые (auth-guard)

| Маршрут                                | Guard             |
| -------------------------------------- | ----------------- |
| `/[locale]/profile`                    | auth              |
| `/[locale]/profile/edit`               | auth              |
| `/[locale]/projects/new`               | auth              |
| `/[locale]/projects/my`                | auth              |
| `/[locale]/projects/[id]/edit`         | owner (`isOwner`) |
| `/[locale]/projects/[id]/applications` | owner             |
| `/[locale]/applications`               | auth (мои заявки) |

Guards: `components/auth/auth-guard.tsx` (по `useAuth().isAuthenticated`), `components/guards/owner-guard.tsx` (по `project.isOwner` из API).
