## Заметки по ключевым компонентам

- **apply-button** — обрабатывает 401 (→ login с returnUrl), 409 («уже откликался»), 422 (message из ErrorResponse); заявка требует CV-вложение
- **position-card** — для owner'а кнопки Close / Reopen (отдельные API, НЕ toggle isFilled)
- **rich-text-editor** — TipTap; выдаёт HTML для `FullDescription`; вывод через `rich-text-content` (backend санитизирует HtmlSanitizer'ом)
- **auth-guard** — client component, редиректит по `useAuth().isAuthenticated`; показывает loading пока `isLoading`
- **owner-guard** — по `project.isOwner` из API (не по локальному сравнению id)
- **technology-\*** в ui/ — мультивыбор технологий + создание новой (POST `/api/lookups/technologies`)
